# Setup

## Installation

Install xldx using your preferred package manager:

```bash
# npm
npm install xldx

# bun
bun add xldx

# yarn
yarn add xldx

# pnpm
pnpm add xldx
```

## Browser Usage

```typescript
import { Xldx } from 'xldx/browser';

const data = [
  { id: 1, name: 'Item 1', value: 100 },
  { id: 2, name: 'Item 2', value: 200 }
];

const xldx = new Xldx(data);
xldx.createSheet(
  { name: 'Sheet1' },
  { key: 'id', header: 'ID', width: 10 },
  { key: 'name', header: 'Name', width: 20 },
  { key: 'value', header: 'Value', width: 15 }
);

// Download file directly
await xldx.download('data.xlsx');

// Or get as Blob
const blob = await xldx.toBlob();
```

## Node.js Usage

```typescript
import { Xldx } from 'xldx/server';

const data = [...];

const xldx = new Xldx(data);
xldx.createSheet(options, ...columns);

// Write to file
await xldx.write('output.xlsx');

// Or get as Buffer
const buffer = await xldx.toBuffer();
```

## TypeScript Configuration

xldx is written in TypeScript and includes type definitions. No additional configuration is needed for TypeScript projects.

```json
{
  "compilerOptions": {
    "moduleResolution": "node",
    "esModuleInterop": true
  }
}
```
