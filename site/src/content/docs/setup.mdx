# Setup

## Installation

Install xldx using your preferred package manager:

<CodeBlock code={`# npm
npm install xldx

# bun
bun add xldx

# yarn
yarn add xldx

# pnpm
pnpm add xldx`} lang="bash" />

## Browser Usage

<CodeBlock code={`import { Xldx } from 'xldx/browser';

const data = [
  { id: 1, name: 'Item 1', value: 100 },
  { id: 2, name: 'Item 2', value: 200 }
];

const xldx = new Xldx(data);
xldx.createSheet(
  { name: 'Sheet1' },
  { key: 'id', header: 'ID', width: 10 },
  { key: 'name', header: 'Name', width: 20 },
  { key: 'value', header: 'Value', width: 15 }
);

// Download file directly
await xldx.download('data.xlsx');

// Or get as Blob
const blob = await xldx.toBlob();`} />

## node, bun, or deno Usage

<CodeBlock code={`import { Xldx } from 'xldx/server';

const data = [...];

const xldx = new Xldx(data);
xldx.createSheet(options, ...columns);

// Write to file
await xldx.write('output.xlsx');

// Or get as Buffer
const buffer = await xldx.toBuffer();`} />

## Type Definitions

xldx includes full type definitions for excellent IDE support. No additional configuration needed.
