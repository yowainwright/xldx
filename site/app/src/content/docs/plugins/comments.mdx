# Comments Plugin

Add comments and notes to Excel cells.

## Installation

```bash
bun add @xldx/comments
```

## Usage

```typescript
import { Xldx } from 'xldx';
import { commentsPlugin, addComment } from '@xldx/comments';

const plugin = commentsPlugin();

const xldx = new Xldx(data)
  .use(plugin)
  .createSheet({ name: 'Report' }, ...columns);

addComment(plugin, {
  cell: 'A1',
  author: 'System',
  text: 'This value was auto-calculated'
});

addComment(plugin, {
  cell: 'B2',
  author: 'Reviewer',
  text: 'Please verify this data'
});

const xlsx = await xldx.toUint8Array();
```

## API

### `commentsPlugin()`

Creates a new comments plugin instance.

### `addComment(plugin, comment)`

Adds a comment to a cell.

```typescript
interface Comment {
  cell: string;      // Cell reference (e.g., "A1", "B2")
  author: string;    // Comment author name
  text: string;      // Comment text content
  sheet?: string;    // Optional: sheet name (defaults to first sheet)
}
```

### Plugin Methods

The plugin instance also exposes direct methods:

```typescript
// Add a comment directly
plugin.addComment({
  cell: 'A1',
  author: 'System',
  text: 'Comment text'
});

// Get all comments
const comments = plugin.getComments();
```

## Examples

### Review Comments

```typescript
const plugin = commentsPlugin();

addComment(plugin, {
  cell: 'B5',
  author: 'Auditor',
  text: 'Verified against source documentation'
});

addComment(plugin, {
  cell: 'C10',
  author: 'Manager',
  text: 'Approved for Q4 reporting'
});
```

### Automated Notes

```typescript
data.forEach((row, index) => {
  if (row.status === 'pending') {
    addComment(plugin, {
      cell: `A${index + 2}`,
      author: 'System',
      text: `Created: ${row.createdAt}\nLast updated: ${row.updatedAt}`
    });
  }
});
```
